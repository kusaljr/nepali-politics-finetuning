# How to Cite This Project

Thank you for using this repository! If you use this project in your research, please cite it using one of the formats below.

## BibTeX (Copy & Paste)

```bibtex
@software{kusal_nepali_finetuning_2026,
  author       = {Kusal Junior},
  title        = {Nepali Politics Fine-tuning: Fine-tuning Gemma-3-270M on Nepali Election Q&A},
  year         = {2026},
  url          = {https://github.com/kusaljr/nepali-politics-finetuning},
  version      = {1.0.0},
  license      = {MIT}
}
```

## APA Format (Copy & Paste)

```
Junior, K. (2026). *Nepali politics fine-tuning: Fine-tuning Gemma-3-270M on Nepali election Q&A* 
[Software]. https://github.com/kusaljr/nepali-politics-finetuning
```

## MLA Format (Copy & Paste)

```
Junior, Kusal. "Nepali Politics Fine-tuning: Fine-tuning Gemma-3-270M on Nepali Election Q&A." 
GitHub, 16 June 2026, github.com/kusaljr/nepali-politics-finetuning.
```

## Chicago Format (Copy & Paste)

```
Junior, Kusal. "Nepali Politics Fine-tuning: Fine-tuning Gemma-3-270M on Nepali Election Q&A." 
Accessed [Access Date]. https://github.com/kusaljr/nepali-politics-finetuning.
```

## Harvard Format (Copy & Paste)

```
Junior, K., 2026. Nepali politics fine-tuning: Fine-tuning Gemma-3-270M on Nepali election Q&A. 
[online] Available at: https://github.com/kusaljr/nepali-politics-finetuning
```

## Automated Citation (GitHub)

GitHub will automatically detect the `CITATION.cff` file. Researchers can:

1. Click the **"Cite this repository"** button (⭐ star icon) on the repository homepage
2. Copy the citation in their preferred format
3. Paste directly into their paper
